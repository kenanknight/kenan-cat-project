# kenan-cat-project
 Cat Photo App

A simple multi-page HTML website about cats, built as practice while following freeCodeCamp and expanding on the project with my own structure, links, and pages.

## About

This project helped me practice basic HTML structure, including:

- Headings and paragraphs
- Images
- Links
- Ordered and unordered lists
- Multiple HTML pages
- Basic page navigation
- Footer sections
- Git and GitHub workflow

## Pages

- Home page
- Things cats love
- Things cats hate
- Thank you page

## What I Learned

While building this project, I practiced creating files, organizing a small website, linking pages together, and using Git commands to track and push my progress.

## Credit

This project was inspired by and based on lessons from [freeCodeCamp](https://www.freecodecamp.org/).

Additional page structure and content were added for extra practice.
